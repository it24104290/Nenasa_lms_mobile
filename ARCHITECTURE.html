📋 System Components

Frontend Layer (Netlify)

  • URL: https://nenasamobile.netlify.app
  • Framework: React 18.3.1 with Vite 6.0.0
  • Features: SPA routing, auto-deploy from GitHub, responsive design
  • UI Framework: Tailwind CSS 4.2.2
  • Charts: Recharts 3.8.0
  • Navigation: NavBar (desktop) + BottomNavBar (mobile)

Backend API Layer (Render)

  • URL: https://nenasa-lms-mobile.onrender.com
  • Runtime: Node.js v20.19.6
  • Framework: Express.js 4.19.2
  • Authentication: JWT (7-day expiry) + bcryptjs
  • Authorization: Role-based middleware (ADMIN, TEACHER, STUDENT, PAYMENT_OFFICER)
  • File Upload: Multer disk storage
  • CORS: Configured for Netlify frontend

Database Layer (MongoDB Atlas)

  • Service: MongoDB Atlas (Cloud)
  • Database: nanasa_lms
  • Collections: Single app_state document with all data
  • Data Models: Users, Teachers, Students, Classes, Lessons, Exams, Payments, Feedback, Analytics
  • Hybrid Support: Falls back to local JSON if MongoDB unavailable

🔐 Security & Authentication

  • Password Hashing: bcryptjs (10 rounds)
  • Token-based Auth: JWT bearer tokens with 7-day expiry
  • Authorization: Role-based access control middleware
  • HTTPS: All communication encrypted
  • CORS: Restricted to authorized frontend origin

🚀 Deployment

  • Version Control: GitHub (it24104290/Nenasa_lms_mobile)
  • Frontend Hosting: Netlify (auto-deploy on push to main)
  • Backend Hosting: Render (auto-deploy on push to main)
  • Database: MongoDB Atlas (cloud-hosted)
  • CI/CD: Automatic from GitHub

👥 User Roles & Permissions

ADMIN

- Manage all users

- View system analytics

- System configuration

TEACHER

- Create classes

- Upload lessons

- Create & grade exams

STUDENT

- Enroll in classes

- Take exams

- View lessons

PAYMENT_OFFICER

- Approve/decline payments

- View transactions

- Manage fees

📊 Key Features

  • Authentication: Secure login/registration with JWT
  • Class Management: Teachers create and manage classes
  • Lesson Management: Upload videos, PDFs, notes
  • Exam Engine: MCQ auto-grading, essay submissions
  • Payment System: Fee collection with approval workflow
  • Analytics: Teacher performance, pass rates, student rankings
  • Feedback System: Anonymous and named feedback
  • Mobile Responsive: Full mobile support with bottom navigation

📁 Project Structure

  • frontend/ - React application with pages, components, services
  • backend/ - Node.js/Express API server with routes, middleware, database adapter
  • GitHub: Version control and CI/CD
  • MongoDB Atlas: Cloud database
